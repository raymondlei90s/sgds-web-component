Description list component for key-value pairs.

<img src="/images/components/descriptionlist/default.png" alt="description-list component preview" />

## Basic Usage

```html
<sgds-description-list>
  <sgds-description-item term="Name" description="John Doe"></sgds-description-item>
  <sgds-description-item term="Email" description="john@example.com"></sgds-description-item>
</sgds-description-list>
```

## API Reference

### Properties
| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `term` | `string` | - | Term/key |
| `description` | `string` | - | Description/value |
