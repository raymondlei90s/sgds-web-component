Icon component for displaying icons.

## Basic Usage

```html
<sgds-icon name="home"></sgds-icon>
<sgds-icon name="user" size="lg"></sgds-icon>
```

## API Reference

### Properties
| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `name` | `string` | - | Icon name |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Icon size |
