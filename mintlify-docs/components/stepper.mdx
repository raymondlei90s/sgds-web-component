Stepper component for multi-step processes.

## Basic Usage

```html
<sgds-stepper activeIndex="0">
  <sgds-stepper-item>Step 1</sgds-stepper-item>
  <sgds-stepper-item>Step 2</sgds-stepper-item>
  <sgds-stepper-item>Step 3</sgds-stepper-item>
</sgds-stepper>
```

## API Reference

### Properties
| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `activeIndex` | `number` | `0` | Current active step |
