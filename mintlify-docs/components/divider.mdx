Divider component for visual separation.

## Basic Usage

```html
<sgds-divider></sgds-divider>
<sgds-divider variant="dashed"></sgds-divider>
```

## API Reference

### Properties
| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `'solid' \| 'dashed'` | `'solid'` | Divider style |
