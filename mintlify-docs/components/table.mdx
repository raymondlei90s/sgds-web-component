Data table component.

## Basic Usage

```html
<sgds-table
  rowHeader='["Name", "Email", "Role"]'
  tableData='[{"name": "John", "email": "john@example.com", "role": "Admin"}]'
></sgds-table>
```

## API Reference

### Properties
| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `rowHeader` | `array` | - | Table headers |
| `tableData` | `array` | - | Table data |
| `tableClasses` | `string` | - | Additional CSS classes |
