Tooltip component for contextual help.

<img src="/images/components/tooltip/default.png" alt="tooltip component preview" />

## Basic Usage

```html
<sgds-tooltip content="Helpful tooltip text">
  <sgds-button>Hover me</sgds-button>
</sgds-tooltip>
```

## API Reference

### Properties
| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `content` | `string` | - | Tooltip content |
| `placement` | `'top' \| 'bottom' \| 'left' \| 'right'` | `'top'` | Tooltip position |
