The input component provides accessible text input with built-in validation support.

## Basic Usage

```html
<sgds-input label="Email" type="email" placeholder="your.email@example.com"></sgds-input>
<sgds-input label="Password" type="password"></sgds-input>
<sgds-input label="Name" required></sgds-input>
```

## Input Types

```html
<sgds-input type="text" label="Text"></sgds-input>
<sgds-input type="email" label="Email"></sgds-input>
<sgds-input type="password" label="Password"></sgds-input>
<sgds-input type="number" label="Number"></sgds-input>
<sgds-input type="tel" label="Phone"></sgds-input>
<sgds-input type="url" label="URL"></sgds-input>
<sgds-input type="search" label="Search"></sgds-input>
```

## States

```html
<sgds-input label="Disabled" disabled></sgds-input>
<sgds-input label="Read Only" readonly value="Cannot edit"></sgds-input>
<sgds-input label="Required" required></sgds-input>
```

## Validation & Feedback

```html
<sgds-input
  label="Email"
  type="email"
  required
  invalidFeedback="Please enter a valid email"
  hasFeedback
></sgds-input>

<sgds-input
  label="Username"
  pattern="[a-zA-Z0-9]+"
  invalidFeedback="Only alphanumeric characters allowed"
  hasFeedback
></sgds-input>
```

## Help Text

```html
<sgds-input label="Password" type="password">
  <span slot="feedback">Must be at least 8 characters</span>
</sgds-input>
```

## React Example

```jsx
import { useState } from 'react';
import { SgdsInput } from '@govtechsg/sgds-web-component/react';

function LoginForm() {
  const [email, setEmail] = useState('');

  return (
    <SgdsInput
      label="Email"
      type="email"
      value={email}
      onSgdsInput={(e) => setEmail(e.target.value)}
      required
    />
  );
}
```

## API Reference

### Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `label` | `string` | - | Input label |
| `type` | `string` | `'text'` | Input type |
| `value` | `string` | `''` | Input value |
| `placeholder` | `string` | - | Placeholder text |
| `disabled` | `boolean` | `false` | Disables input |
| `readonly` | `boolean` | `false` | Makes input read-only |
| `required` | `boolean` | `false` | Makes input required |
| `pattern` | `string` | - | Validation pattern |
| `invalidFeedback` | `string` | - | Error message |
| `hasFeedback` | `boolean` | `false` | Shows validation feedback |

### Events

| Event | Description |
|-------|-------------|
| `sgds-input` | Fired on input (real-time) |
| `sgds-change` | Fired on change (on blur) |
| `sgds-focus` | Fired when input receives focus |
| `sgds-blur` | Fired when input loses focus |

### Slots

| Slot | Description |
|------|-------------|
| `feedback` | Help text or validation message |
| `icon-left` | Icon before input |
| `icon-right` | Icon after input |
