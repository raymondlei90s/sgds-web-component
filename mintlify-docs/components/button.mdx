The button component provides consistent, accessible buttons for user actions.

## Basic Usage

```html
<sgds-button>Default Button</sgds-button>
<sgds-button variant="primary">Primary Button</sgds-button>
<sgds-button variant="secondary">Secondary Button</sgds-button>
```

## Framework Examples

<Tabs>
  <Tab title="React">
    ```jsx
    import { SgdsButton } from '@govtechsg/sgds-web-component/react';

    function App() {
      const handleClick = () => {
        console.log('Button clicked!');
      };

      return (
        <SgdsButton variant="primary" onClick={handleClick}>
          Click Me
        </SgdsButton>
      );
    }
    ```
  </Tab>

## API Reference

### Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `'solid' \| 'primary' \| 'secondary' \| 'outline' \| 'outline-primary' \| 'ghost' \| 'link'` | `'solid'` | Button visual style |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Button size |
| `disabled` | `boolean` | `false` | Disables the button |
| `loading` | `boolean` | `false` | Shows loading state |
| `type` | `'button' \| 'submit' \| 'reset'` | `'button'` | Button type attribute |
| `href` | `string` | - | Makes button behave as link |

### Events

| Event | Description |
|-------|-------------|
| `click` | Fired when button is clicked |
| `focus` | Fired when button receives focus |
| `blur` | Fired when button loses focus |

### Slots

| Slot | Description |
|------|-------------|
| `default` | Button text content |
| `icon-left` | Icon before text |
| `icon-right` | Icon after text |

### CSS Custom Properties

| Property | Description |
|----------|-------------|
| `--button-bg` | Background color |
| `--button-color` | Text color |
| `--button-border-color` | Border color |
| `--button-padding-x` | Horizontal padding |
| `--button-padding-y` | Vertical padding |

### CSS Parts

| Part | Description |
|------|-------------|
| `base` | The button element |
| `content` | Button content wrapper |

## Accessibility

- Uses native `<button>` element
- Supports keyboard navigation (Enter, Space)
- Proper ARIA attributes for disabled/loading states
- Focus visible styles
