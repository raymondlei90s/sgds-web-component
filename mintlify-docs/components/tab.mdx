Tab component for organizing content.

## Basic Usage

```html
<sgds-tab-group>
  <sgds-tab active>Tab 1</sgds-tab>
  <sgds-tab>Tab 2</sgds-tab>
  <sgds-tab>Tab 3</sgds-tab>
</sgds-tab-group>
```

## API Reference

### Properties
| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `active` | `boolean` | `false` | Active tab |

### Events
| Event | Description |
|-------|-------------|
| `sgds-tab-change` | Fired when tab changes |
