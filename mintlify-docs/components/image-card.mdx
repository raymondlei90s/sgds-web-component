Card component with image header.

## Basic Usage

```html
<sgds-image-card>
  <img slot="image" src="/image.jpg" alt="Card image" />
  <span slot="title">Card Title</span>
  <p slot="description">Card description</p>
  <a slot="link" href="/read-more">Read More</a>
</sgds-image-card>
```

## API Reference

### Slots
| Slot | Description |
|------|-------------|
| `image` | Card image |
| `title` | Card title |
| `description` | Card description |
| `link` | Card link |
