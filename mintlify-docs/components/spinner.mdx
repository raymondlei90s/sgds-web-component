Loading spinner component.

## Basic Usage

```html
<sgds-spinner></sgds-spinner>
<sgds-spinner size="sm"></sgds-spinner>
<sgds-spinner size="lg"></sgds-spinner>
```

## API Reference

### Properties
| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Spinner size |
