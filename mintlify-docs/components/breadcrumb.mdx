## Basic Usage

```html
<sgds-breadcrumb>
  <sgds-breadcrumb-item href="/">Home</sgds-breadcrumb-item>
  <sgds-breadcrumb-item href="/products">Products</sgds-breadcrumb-item>
  <sgds-breadcrumb-item active>Current Page</sgds-breadcrumb-item>
</sgds-breadcrumb>
```

## API Reference

### Properties
| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `active` | `boolean` | `false` | Active breadcrumb item |
