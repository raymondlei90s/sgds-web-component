<img src="/images/components/link/default.png" alt="link component preview" />

## Basic Usage

```html
<sgds-link href="/about">About Us</sgds-link>
<sgds-link href="/contact" external>Contact (opens in new tab)</sgds-link>
```

## API Reference

### Properties
| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `href` | `string` | - | Link URL |
| `external` | `boolean` | `false` | Open in new tab |
| `disabled` | `boolean` | `false` | Disabled state |
