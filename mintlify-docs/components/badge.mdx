Badge component for status indicators.

## Basic Usage

```html
<sgds-badge variant="primary">New</sgds-badge>
<sgds-badge variant="success">Active</sgds-badge>
<sgds-badge variant="warning">Pending</sgds-badge>
<sgds-badge variant="danger">Error</sgds-badge>
```

## API Reference

### Properties
| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `string` | `'default'` | Badge variant |
