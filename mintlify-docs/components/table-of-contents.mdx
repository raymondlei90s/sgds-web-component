Table of contents component for in-page navigation.

## Basic Usage

```html
<sgds-table-of-contents></sgds-table-of-contents>
```

## API Reference

### Properties
| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `selector` | `string` | `'h2, h3'` | Headings to include |
