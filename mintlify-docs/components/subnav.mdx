Secondary navigation component for subsections.

## Basic Usage

```html
<sgds-subnav>
  <sgds-subnav-item href="/section1">Section 1</sgds-subnav-item>
  <sgds-subnav-item href="/section2" active>Section 2</sgds-subnav-item>
  <sgds-subnav-item href="/section3">Section 3</sgds-subnav-item>
</sgds-subnav>
```

## API Reference

### Properties
| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `active` | `boolean` | `false` | Active state |
