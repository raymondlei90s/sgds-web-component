System-wide banner for announcements.

## Basic Usage

```html
<sgds-system-banner show>
  Important system announcement
</sgds-system-banner>
```

## API Reference

### Properties
| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `show` | `boolean` | `false` | Visibility state |
| `closable` | `boolean` | `false` | Show close button |
