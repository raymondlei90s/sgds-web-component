Progress bar component.

<img src="/images/components/progressbar/default.png" alt="progress-bar component preview" />

## Basic Usage

```html
<sgds-progress-bar value="50"></sgds-progress-bar>
<sgds-progress-bar value="75" variant="success"></sgds-progress-bar>
```

## API Reference

### Properties
| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `value` | `number` | `0` | Progress value (0-100) |
| `variant` | `string` | `'default'` | Bar variant |
