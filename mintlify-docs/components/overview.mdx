## Quick Start

All components follow the same pattern:

```html
<!-- Import once in your app -->
<script src="https://cdn.jsdelivr.net/npm/@govtechsg/sgds-web-component@3.18.1"></script>

<!-- Use anywhere -->
<sgds-button variant="primary">Click Me</sgds-button>
<sgds-input label="Email" type="email"></sgds-input>
<sgds-alert show variant="success">Success!</sgds-alert>
```

## Framework Support

<Tabs>
  <Tab title="React">
    ```jsx

    function App() {
      return (
        <>

          <SgdsInput label="Email" type="email" />

        </>
      );
    }
    ```
  </Tab>

  <Tab title="Vue">
    ```vue

      <sgds-button variant="primary">Click Me</sgds-button>

      <sgds-alert show variant="success">Success!</sgds-alert>
    </template>
    ```

## Component Features

All SGDS components share these features:

- ✅ **Accessible**: Built with ARIA best practices
- ✅ **Themeable**: Support for day/night mode
- ✅ **Customizable**: CSS parts and custom properties
- ✅ **Event-driven**: Custom events for framework integration
- ✅ **Typed**: Full TypeScript support
- ✅ **Tested**: Comprehensive test coverage
