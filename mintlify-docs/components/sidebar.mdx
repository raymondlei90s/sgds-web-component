Sidebar layout component for dashboard interfaces.

## Basic Usage

```html
<sgds-sidebar>
  <div slot="sidebar">Sidebar content</div>
  <div slot="content">Main content</div>
</sgds-sidebar>
```

## API Reference

### Slots
| Slot | Description |
|------|-------------|
| `sidebar` | Sidebar content |
| `content` | Main content area |
