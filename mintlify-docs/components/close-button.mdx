Standardized close button for dialogs and dismissible components.

## Basic Usage

```html
<sgds-close-button></sgds-close-button>
```

## API Reference

### Properties
| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `string` | `'default'` | Button variant |

### Events
| Event | Description |
|-------|-------------|
| `click` | Fired when clicked |
