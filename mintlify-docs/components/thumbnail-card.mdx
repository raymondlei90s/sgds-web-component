Compact card with thumbnail image.

## Basic Usage

```html
<sgds-thumbnail-card>
  <img slot="thumbnail" src="/thumb.jpg" alt="Thumbnail" />
  <span slot="title">Item Title</span>
  <p slot="description">Brief description</p>
</sgds-thumbnail-card>
```

## API Reference

### Slots
| Slot | Description |
|------|-------------|
| `thumbnail` | Thumbnail image |
| `title` | Card title |
| `description` | Card description |
