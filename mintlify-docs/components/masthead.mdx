---

The masthead displays official Singapore Government branding at the top of pages.

## Basic Usage

```html
<sgds-masthead></sgds-masthead>
```

## With Fluid Container

```html
<sgds-masthead fluid></sgds-masthead>
```

## API Reference

### Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `fluid` | `boolean` | `false` | Makes masthead full width |

### CSS Custom Properties

| Property | Description |
|----------|-------------|
| `--masthead-bg` | Background color |
| `--masthead-color` | Text color |
