Basic card component for content organization.

<img src="/images/components/card/default.png" alt="card component preview" />

## Basic Usage

```html
<sgds-card>
  <span slot="title">Card Title</span>
  <p slot="description">Card description goes here</p>
  <a slot="link" href="/learn-more">Learn More</a>
</sgds-card>
```

## API Reference

### Slots
| Slot | Description |
|------|-------------|
| `title` | Card title |
| `description` | Card description |
| `link` | Card link |
| `default` | Card body content |

### CSS Parts
| Part | Description |
|------|-------------|
| `card-body` | Card body element |
