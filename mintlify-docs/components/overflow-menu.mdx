Overflow menu for compact action lists.

## Basic Usage

```html
<sgds-overflow-menu>
  <sgds-dropdown-item>Edit</sgds-dropdown-item>
  <sgds-dropdown-item>Delete</sgds-dropdown-item>
  <sgds-dropdown-item>Share</sgds-dropdown-item>
</sgds-overflow-menu>
```

### Slots
| Slot | Description |
