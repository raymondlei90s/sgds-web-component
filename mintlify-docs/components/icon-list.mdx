List component with icon markers.

## Basic Usage

```html
<sgds-icon-list>
  <sgds-icon-list-item>
    <svg slot="icon" xmlns="http://www.w3.org/2000/svg" width="16" height="16" fill="currentColor">
      <path d="..."/>
    </svg>
    List item text
  </sgds-icon-list-item>
</sgds-icon-list>
```

## API Reference

### Slots
| Slot | Description |
|------|-------------|
| `icon` | Item icon |
| `default` | Item content |
