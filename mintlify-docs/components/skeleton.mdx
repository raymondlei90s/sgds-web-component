Skeleton loading placeholder component.

<img src="/images/components/skeleton/default.png" alt="skeleton component preview" />

## Basic Usage

```html
<sgds-skeleton></sgds-skeleton>
<sgds-skeleton width="200px" height="20px"></sgds-skeleton>
```

## API Reference

### Properties
| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `width` | `string` | `'100%'` | Skeleton width |
| `height` | `string` | `'1rem'` | Skeleton height |
