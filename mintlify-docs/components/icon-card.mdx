## Basic Usage

```html
<sgds-icon-card>
  <svg slot="icon" xmlns="http://www.w3.org/2000/svg" width="24" height="24" fill="currentColor">
    <path d="..."/>
  </svg>
  <span slot="title">Feature Title</span>
  <p slot="description">Feature description</p>
</sgds-icon-card>
```

## API Reference

### Slots
| Slot | Description |
|------|-------------|
| `icon` | Card icon |
| `title` | Card title |
| `description` | Card description |
