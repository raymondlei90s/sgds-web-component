Footer component with links and branding.

## Basic Usage

```html
<sgds-footer
  title="Singapore Government Design System"
  description="Build better government services"
  lastUpdatedDate="Last Updated: 1 Jan 2025"
></sgds-footer>
```

## API Reference

### Properties
| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `title` | `string` | - | Footer title |
| `description` | `string` | - | Footer description |
| `lastUpdatedDate` | `string` | - | Last updated date |
| `links` | `array` | - | Footer links |

### CSS Parts
| Part | Description |
|------|-------------|
| `footer-bottom` | Bottom footer section |
