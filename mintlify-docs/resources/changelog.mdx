## Latest Version: v3.18.1

View the complete changelog on [GitHub](https://github.com/GovTechSG/sgds-web-component/blob/master/CHANGELOG.md).

## Major Releases

### v3.0.0 - Design System Overhaul

- Complete design refresh moving away from Bootstrap
- New CSS design token system aligned with Figma
- Built-in dark mode (`night.css` theme)
- Tailwind v4 utilities with `sgds:` prefix
- Breaking changes - see [Migration Guide](/getting-started/migration-guide)

### v2.0.0 - Architecture Update

- Upgraded to Lit v3
- Removed Bootstrap dependency
- Individual component styles
- Global CSS token system
- Improved bundle sizes

### v1.0.0 - Initial Stable Release

- 47 production-ready components
- React wrapper components
- Framework integration guides
- Comprehensive documentation
