## What is SGDS Web Components?

SGDS Web Components is Singapore Government's official design system implementation using native Web Components. Built with [Lit](https://lit.dev), these components work seamlessly across all modern frameworks including React, Vue, Angular, and vanilla JavaScript.

## Quick Example

Get started in seconds with CDN or local installation:

<CodeGroup>
```html CDN
<!DOCTYPE html>
<html>
<head>
  <script src="https://cdn.jsdelivr.net/npm/@govtechsg/sgds-web-component@3.18.1"></script>
</head>
<body>
  <sgds-button variant="primary">Click Me</sgds-button>
  <sgds-alert show variant="success">
    Welcome to SGDS Web Components!
  </sgds-alert>
</body>
</html>
```

```bash npm
npm install @govtechsg/sgds-web-component
```

```javascript import
import "@govtechsg/sgds-web-component";

// Use in your HTML
<sgds-button variant="primary">Click Me</sgds-button>
```
</CodeGroup>

## Who Uses SGDS?

SGDS Web Components are used across Singapore Government digital services, ensuring consistency and accessibility for millions of citizens.

## Browser Support

SGDS Web Components work in all modern browsers:

- ✅ Chrome/Edge 90+
- ✅ Firefox 88+
- ✅ Safari 15+
- ✅ Mobile browsers (iOS Safari, Chrome Mobile)

<Note>
For legacy browser support, you may need the [scoped custom element registry polyfill](https://github.com/webcomponents/polyfills/tree/master/packages/scoped-custom-element-registry).
</Note>
