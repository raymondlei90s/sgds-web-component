Application shell for dashboards and internal tools with sidebar navigation.

## Complete Layout

```html
<body>
  <sgds-masthead></sgds-masthead>

  <div class="sgds:flex">
    <!-- Sidebar -->
    <sgds-sidenav class="sgds:w-64">
      <sgds-sidenav-item>
        <span slot="title">Dashboard</span>
      </sgds-sidenav-item>
      <sgds-sidenav-item>
        <span slot="title">Reports</span>
        <sgds-sidenav-link href="/reports/monthly">Monthly</sgds-sidenav-link>
        <sgds-sidenav-link href="/reports/quarterly">Quarterly</sgds-sidenav-link>
      </sgds-sidenav-item>
      <sgds-sidenav-item>
        <span slot="title">Settings</span>
      </sgds-sidenav-item>
    </sgds-sidenav>

    <!-- Main Content -->
    <main class="sgds:flex-1 sgds:p-8">
      <h1>Dashboard</h1>
      <!-- Content -->
    </main>
  </div>
</body>
```
