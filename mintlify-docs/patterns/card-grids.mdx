Card grid layouts for displaying multiple items.

## 3-Column Grid

```html
<div class="sgds:grid sgds:grid-cols-1 md:sgds:grid-cols-3 sgds:gap-6 sgds:py-16">
  <sgds-card>
    <span slot="title">Card 1</span>
    <p slot="description">Card description</p>
  </sgds-card>
  <sgds-card>
    <span slot="title">Card 2</span>
    <p slot="description">Card description</p>
  </sgds-card>
  <sgds-card>
    <span slot="title">Card 3</span>
    <p slot="description">Card description</p>
  </sgds-card>
</div>
```
