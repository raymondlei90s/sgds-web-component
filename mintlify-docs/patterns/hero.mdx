Hero sections create impactful page headers with headings, descriptions, and call-to-action buttons.

## Variants

### Basic Hero

```html
<div class="sgds:bg-primary-default sgds:text-white sgds:py-16">
  <div class="sgds:max-w-7xl sgds:mx-auto sgds:px-4">
    <h1 class="sgds:text-5xl sgds:font-bold sgds:mb-4">Page Title</h1>
    <p class="sgds:text-xl sgds:mb-6">Compelling description of the page content</p>
    <sgds-button variant="secondary">Get Started</sgds-button>
  </div>
</div>
```

### Center Aligned

```html
<div class="sgds:bg-surface-default sgds:py-20">
  <div class="sgds:max-w-4xl sgds:mx-auto sgds:text-center sgds:px-4">
    <h1 class="sgds:text-5xl sgds:font-bold sgds:mb-4">Welcome</h1>
    <p class="sgds:text-xl sgds:text-subdued sgds:mb-8">
      Start your journey with us today
    </p>
    <div class="sgds:flex sgds:gap-4 sgds:justify-center">
      <sgds-button variant="primary">Primary Action</sgds-button>
      <sgds-button variant="outline">Secondary Action</sgds-button>
    </div>
  </div>
</div>
```

## Best Practices

<AccordionGroup>
  <Accordion title="Keep Headlines Concise">
    Use 5-10 words for hero headlines for maximum impact
  </Accordion>

  <Accordion title="Single Clear CTA">
    Focus on one primary call-to-action with optional secondary action
  </Accordion>

  <Accordion title="Accessible Contrast">
    Ensure text has sufficient contrast against background images/colors
  </Accordion>

  <Accordion title="Responsive Design">
    Test on mobile - reduce text sizes and padding appropriately
  </Accordion>
</AccordionGroup>
