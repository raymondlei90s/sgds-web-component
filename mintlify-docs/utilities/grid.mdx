SGDS provides grid utilities for creating responsive layouts using the `sgds:` prefix.

## Grid Container

```html
<div class="sgds:grid sgds:grid-cols-3 sgds:gap-4">
  <div>Column 1</div>
  <div>Column 2</div>
  <div>Column 3</div>
</div>
```

## Grid Columns

```html
<div class="sgds:grid sgds:grid-cols-1">1 column</div>
<div class="sgds:grid sgds:grid-cols-2">2 columns</div>
<div class="sgds:grid sgds:grid-cols-3">3 columns</div>
<div class="sgds:grid sgds:grid-cols-4">4 columns</div>
<div class="sgds:grid sgds:grid-cols-6">6 columns</div>
<div class="sgds:grid sgds:grid-cols-12">12 columns</div>
```

## Column Span

```html
<div class="sgds:grid sgds:grid-cols-12 sgds:gap-4">
  <div class="sgds:col-span-6">Spans 6 columns</div>
  <div class="sgds:col-span-3">Spans 3 columns</div>
  <div class="sgds:col-span-3">Spans 3 columns</div>
</div>
```

## Responsive Grids

```html
<div class="sgds:grid sgds:grid-cols-1 md:sgds:grid-cols-2 lg:sgds:grid-cols-4 sgds:gap-6">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
  <div>Item 4</div>
</div>
```

## Auto-Fit Columns

```html
<div class="sgds:grid" style="grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); gap: 1.5rem;">
  <div>Auto-sizing column</div>
  <div>Auto-sizing column</div>
  <div>Auto-sizing column</div>
</div>
```
