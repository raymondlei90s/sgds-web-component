SGDS supports automatic dark mode through the built-in night theme.

## Setup

Import both day and night theme files:

```css
/* main.css */
@import "@govtechsg/sgds-web-component/themes/day.css";
@import "@govtechsg/sgds-web-component/themes/night.css";
```

## Activating Dark Mode

Add the `sgds-theme-night` class to the `<html>` element:

```javascript
// Toggle dark mode
document.documentElement.classList.toggle('sgds-theme-night');

// Enable dark mode
document.documentElement.classList.add('sgds-theme-night');

// Disable dark mode
document.documentElement.classList.remove('sgds-theme-night');
```

## Complete Theme Toggle

```html
<button id="theme-toggle">
  <span id="theme-icon">🌙</span>
  <span id="theme-text">Dark Mode</span>
</button>

<script>
  const toggle = document.getElementById('theme-toggle');
  const icon = document.getElementById('theme-icon');
  const text = document.getElementById('theme-text');

  toggle.addEventListener('click', () => {
    const isDark = document.documentElement.classList.toggle('sgds-theme-night');

    // Update button
    icon.textContent = isDark ? '☀️' : '🌙';
    text.textContent = isDark ? 'Light Mode' : 'Dark Mode';

    // Save preference
    localStorage.setItem('theme', isDark ? 'night' : 'day');
  });

  // Restore saved preference on page load
  const savedTheme = localStorage.getItem('theme');
  if (savedTheme === 'night') {
    document.documentElement.classList.add('sgds-theme-night');
    icon.textContent = '☀️';
    text.textContent = 'Light Mode';
  }
</script>
```

## System Preference Detection

Detect user's system theme preference:

```javascript
// Check system preference
const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches;

// Apply on page load if no saved preference
const savedTheme = localStorage.getItem('theme');
if (!savedTheme && prefersDark) {
  document.documentElement.classList.add('sgds-theme-night');
}

// Listen for system preference changes
window.matchMedia('(prefers-color-scheme: dark)').addEventListener('change', (e) => {
  if (!localStorage.getItem('theme')) {
    if (e.matches) {
      document.documentElement.classList.add('sgds-theme-night');
    } else {
      document.documentElement.classList.remove('sgds-theme-night');
    }
  }
});
```

## Framework Integration

<Tabs>
  <Tab title="React">
    ```jsx
    import { useEffect, useState } from 'react';

    export function ThemeToggle() {
      const [isDark, setIsDark] = useState(false);

      useEffect(() => {
        const saved = localStorage.getItem('theme');
        if (saved === 'night') {
          setIsDark(true);
          document.documentElement.classList.add('sgds-theme-night');
        }
      }, []);

      const toggleTheme = () => {
        setIsDark(!isDark);
        document.documentElement.classList.toggle('sgds-theme-night');
        localStorage.setItem('theme', !isDark ? 'night' : 'day');
      };

      return (
        <button onClick={toggleTheme}>
          {isDark ? '☀️ Light' : '🌙 Dark'}
        </button>
      );
    }
    ```
  </Tab>

    ```vue
    <template>
      <button @click="toggleTheme">
        {{ isDark ? '☀️ Light' : '🌙 Dark' }}
      </button>

    <script>
    import { ref, onMounted } from 'vue';

    export default {
      setup() {
        const isDark = ref(false);

        onMounted(() => {
          const saved = localStorage.getItem('theme');
          if (saved === 'night') {
            isDark.value = true;
            document.documentElement.classList.add('sgds-theme-night');
          }
        });

        const toggleTheme = () => {
          isDark.value = !isDark.value;
          document.documentElement.classList.toggle('sgds-theme-night');
          localStorage.setItem('theme', isDark.value ? 'night' : 'day');
        };

        return { isDark, toggleTheme };
      }
    };
    </script>
    ```

## Theme-Aware Utilities

Use theme-aware utility classes that automatically adapt:

```html
<div class="sgds:bg-surface-default sgds:text-default sgds:border sgds:border-default sgds:p-6">
  This content adapts to the current theme
</div>
```

## Custom Dark Mode Overrides

Customize dark mode colors:

```css
/* your-theme.css */
.sgds-theme-night {
  /* Override specific night theme values */
  --sgds-bg-default: #1a1a1a;
  --sgds-text-default: #f0f0f0;

  /* Adjust primary color for dark mode */
  --sgds-primary-default: var(--sgds-product-primary-400); /* Lighter in dark */
}
```
